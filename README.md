
**Road Accident Analysis in India using Excel and Power BI.**

1. **Data Collection**

   * The dataset used in the project was collected from **Dataful.in** in **Excel** format.
   * Data included **state, year, vehicle type, driver details, road conditions, accident severity**, etc.

2. **Data Cleaning & Preparation**

   * Removed duplicate or incomplete entries in Excel.
   * Standardized values (e.g., vehicle types, road types, accident severity).
   * Ensured data was ready for visualization in Power BI.

3. **Data Analysis in Excel**

   * Calculated **total accidents, casualties, and fatalities**.
   * Analyzed **accidents by state, year, vehicle type, driver gender, and license status**.
   * Checked trends and identified **high-risk roads and conditions**.

4. **Data Visualization in Power BI**

   * Imported cleaned Excel data into **Power BI**.
   * Created **interactive charts and dashboards**:

     * Year-wise trends
     * State-wise accident distribution
     * Vehicle type involvement
     * Accident severity
     * Alcohol involvement
   * Highlighted **high-density accident regions** and other insights.

5. **Insights Extraction**

   * Summarized key findings, such as:

     * Most risky states, roads, and vehicles
     * Male drivers more involved
     * Alcohol contributes to ~50% of accidents
     * Serious/fatal accidents form a large portion

6. **Dashboard Deployment**

   * Built a **fully interactive Power BI dashboard** for easy exploration.
   * Added **filters** for state, year, vehicle type, and road conditions.
